طريقة التشغيل / How to Run
===============================================

🌐 للاستضافة على الويب / For Web Hosting:

1. ارفع جميع الملفات إلى مجلد الاستضافة
   Upload all files to your hosting folder

2. تأكد أن index.html في المجلد الجذر
   Ensure index.html is in the root directory

3. افتح الرابط في المتصفح
   Open the URL in your browser

4. التطبيق جاهز للاستخدام!
   Application is ready to use!

💻 للتشغيل المحلي / For Local Testing:

1. افتح index.html في أي متصفح
   Open index.html in any browser

2. التطبيق سيعمل محلياً
   Application will run locally

===============================================
كيفية الاستخدام / How to Use
===============================================

📋 خطوات الاختبار / Testing Steps:

الخطوة 1: اختيار نوع الاختبار
Step 1: Select Test Type
- اختبار ماركيز / Marquis Test
- اختبار كبريتات الحديد / Ferric Sulfate Test  
- اختبار ميكي / Mecke Test

الخطوة 2: مراجعة التعليمات
Step 2: Review Instructions
- اقرأ تعليمات السلامة / Read safety instructions
- اتبع خطوات التحضير / Follow preparation steps

الخطوة 3: اختيار اللون المُلاحظ
Step 3: Select Observed Color
- اختر اللون الذي تراه / Select the color you observe
- انقر على العينة المناسبة / Click on appropriate sample

الخطوة 4: مراجعة النتائج
Step 4: Review Results
- اطلع على النتيجة / Review the result
- اطبع التقرير إذا لزم الأمر / Print report if needed

استكشاف الأخطاء / Troubleshooting
===============================================

❌ المشكلة: الصفحة لا تفتح
Problem: Page doesn't open

✅ الحل / Solution:
- تأكد من اسم الملف index.html
- تأكد من رفع جميع الملفات
- جرب متصفح مختلف

❌ المشكلة: الوظائف لا تعمل
Problem: Functions don't work

✅ الحل / Solution:
- تأكد من تفعيل JavaScript
- تحقق من وجود أخطاء في Console
- أعد تحميل الصفحة

❌ المشكلة: الطباعة لا تعمل
Problem: Printing doesn't work

✅ الحل / Solution:
- تأكد من وجود print-report.html
- اسمح بالنوافذ المنبثقة
- جرب متصفح مختلف
